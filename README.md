# Regulatory Management
Project for managing regulations for employees who work in activities that require mandatory courses by the Ministry of Labor

## Projects technologies
- [x] Spring Boot 3.1.5
- [x] Spring Web
- [x] Spring Security
- [x] Spring Boot JPA
- [x] Spring Validations
- [x] H2 Database
- [x] Java JWT 4.4.0
